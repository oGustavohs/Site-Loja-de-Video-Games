# LojaVideoGame

Uma pequena loja de videogames desenvolvida como projeto pessoal.  
Interface moderna, responsiva e focada em UX para explorar produtos, com visual inspirado em games clássicos.

## 🚀 Sobre o projeto

- Nome: LojaVideoGame
- Tipo: site estático front-end (HTML, CSS puro, imagens)
- Objetivo: praticar layout responsivo, acessibilidade básica e organização de conteúdo de catálogo.
- Público: fãs de games e quem está aprendendo web dev.

## 🧩 Estrutura

- `index.html` - página principal.
- `style.css` - estilos e responsividade.
- `assets/` - imagens (ex: `TheLegendOfZelda.avif`).

## 🛠️ Tecnologias

- HTML5
- CSS3
- Imagens AVIF (alta qualidade + tamanho reduzido)
